This is the landing page for the Django AdminLTE PDQ package CSS.

The provided stylings should match AdminLTE2, an open-source front-end framework, designed to provide an easy dashboard and control panel theme for any website. All of this runs off of Bootstrap 3.3.7.

Click any of the links in the navbar to the left to see some of the provided package stylings. Or compare to the original AdmninLTE docs to see the full stylings this package is based on.

For examples of using these package styles in Django templates, see the Django AdminLTE PDQ CSS Demo template files. .


Note: The Django AdminLTE PDQ package also slightly expands the AdminLTE framework, to have coloring support for all of bootstrap's built-in colors.
